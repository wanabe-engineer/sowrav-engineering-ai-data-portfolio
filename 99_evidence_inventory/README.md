# Evidence Inventory

This folder contains the audit trail for how the local GitHub-ready portfolio was assembled.

## Files

- `local_file_manifest.csv`: file listing for the curated local portfolio package.
- `GMAIL_DRIVE_CLASSROOM_ANALYSIS.md`: summary of connected Gmail, Drive, and Google Classroom evidence.
- `NICE_TO_HAVES.md`: recommended upgrades before public release.
- `GITHUB_UPLOAD_GUIDE.md`: manual upload instructions that do not require API keys.
