# Evidence Map

Use the parent respiratory AI folder as the private evidence source. Do not publish sensitive clinical, IRB, or unpublished manuscript material without review.

Recommended evidence to curate:

- digital stethoscope reports and presentation materials
- respiratory sound classification drafts
- RIP literature review and project documents
- algorithm or model notes
- progress-report material that can be safely summarized

Public GitHub should contain cleaned diagrams, synthetic examples, methodology, and non-sensitive excerpts only.
