# Profile and Credentials

This folder supports the human story behind the technical portfolio.

- CV and updated CV variants.
- Academic results.
- LinkedIn/profile portfolio PDF.
- Research-fit profile for funded MSc/PhD outreach.

## Portfolio Use

Use this folder as supporting material, not as the first thing a GitHub visitor sees. The main README should lead with projects and research; this folder should validate credentials when a professor, recruiter, or reviewer wants proof.

## Recommended Public Version

For a public repository, create a redacted `Sowrav_Chowdhury_CV_Public.pdf` with:

- Personal phone number removed or replaced with email/LinkedIn.
- References removed unless they have consented.
- Full transcript/results kept private unless required.
- Research and publication links preserved.
