## Portfolio Question

How can an electrical services design be documented from building plan to circuit layout, distribution assumptions, and final deliverable drawings?

## Methodology To Present

1. Define building context and load categories.
2. Identify lighting, socket, fixture, and distribution requirements.
3. Create layout drawings and panel/distribution assumptions.
4. Document design constraints and safety considerations.
5. Present final drawings with a readable design narrative.

## Technical Framing

Estimated connected load:

\[
P_{total} = \sum_i P_i
\]

Single-phase current estimate:

\[
I = \frac{P}{V \cdot \text{pf}}
