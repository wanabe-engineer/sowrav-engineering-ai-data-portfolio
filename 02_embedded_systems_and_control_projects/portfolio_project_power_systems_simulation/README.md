## Methodology To Build

1. Extract experiment objectives and circuit/test setup.
2. Convert measured/simulated values into a structured table.
3. Calculate equivalent parameters or performance metrics.
4. Plot voltage, current, power, efficiency, or power-factor changes.
5. Summarize findings and limitations.

## Technical Framing

Apparent, real, and reactive power:

\[
S = VI,\quad P = VI\cos\phi,\quad Q = VI\sin\phi
\]

Power factor:

\[
\text{pf} = \cos\phi = \frac{P}{S}
\]
